# Java Swing Calculator
A simple, fully functional calculator application developed using the Java Swing GUI toolkit.

## Features
- Basic arithmetic operations: Addition, Subtraction, Multiplication, Division.
- Modulo operation (%).
- Clear functionality.
- Floating-point support.

## Technologies Used
- Java
- Swing (GUI)

## How to Run
1. Clone this repository: `git clone https://github.com/Muhammed-Afsal-T/Java-Swing-Calculator.git`
2. Compile the `Calculator.java` file.
3. Run the compiled class file.
