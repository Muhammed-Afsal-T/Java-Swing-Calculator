## Features
- Basic arithmetic operations: Addition, Subtraction, Multiplication, Division.
- Modulo operation (%).
- Clear functionality.
- High-Precision Calculation.
- Negative Number Support.
- Error Handling.

## Technologies Used
- Java
- Swing (GUI)

## How to Run
1. Clone this repository: `git clone https://github.com/Muhammed-Afsal-T/Java-Swing-Calculator.git`
2. Compile the `Calculator.java` file.
3. Run the compiled class file.
